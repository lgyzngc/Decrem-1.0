This folder contains the main scripts to implement efficient loopless optimization.

For examples of how to run this code, please revise and execute 'Example1_ll_FVA.m' and 'Example2_LD_LR.m'.

*Flux removal is not supported in OCTAVE as glpk cannot solve MIQP problems
